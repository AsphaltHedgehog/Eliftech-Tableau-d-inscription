## How to Run

### Envs:

- env for back: PORT + DBHOST (MongoDB url);
- env for front: VITE_BASE_URL (Backend url);

### Scripts:

1. **Backend:**

- npm run start:dev (development);
- npm run render (production);

2. **Frontend:**

- npm run dev (development);
- npm run build (production);
